🥇 Discovery & Recommendation API

Power smart search and personalized recommendations like Netflix or YouTube.

Key Features

Autocomplete Suggestions

Real-time suggestions while typing with ranking by relevance, popularity, and user interest.

Personalized Recommendations

"Recommended for you" sections based on user behavior and preferences.

Trending Services

Shows what's popular and gaining traction in the marketplace.

Similar Listings

Shows related services on each product page for cross-selling.

API Endpoints

GET /api/discovery/autocomplete?q=search&type=all

Get autocomplete suggestions for search queries

GET /api/discovery/recommendations?limit=10

Get personalized recommendations for current user

GET /api/discovery/trending?limit=10&category=web

Get trending services with optional category filter

GET /api/discovery/similar/{id}?limit=6

Get similar listings for a specific product
